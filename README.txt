"Limopay" - Responsive Wordpress Payment Gateway Plugin

== Description ==

Limopay is an online payment Gateway. From Limopay people can easily  online payment.
People can get payment by WordPress plugin but there must be integrated WooCommerce in theme. We provide the best security that's why here we use git pass authenticator. It required When a customer want to payment using limopay. They used their limopay user name and git pass OTP.  
Make payments and manage money in a secure way.This is payment gateway link https://limopay.net/


==Why choose Limopay==
Limopay gives your customers more flexibility to buy products.



==Requirement==
 1.Create limopay bussiness account from here <a href='https://limopay.net/'><b>click</b></a>
 2.Download Git Pass Apps from google play store.
 3.Git Authenticator enable.
 4.Need WooCommerce Plugin.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Limopay'
3. Activate Limopay from your Plugins page.




= From WordPress.org =

1. Download Limopay Plugin.
2. Upload the 'Limopay' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Bdtask 'Limopay' from your Plugins page.


== Frequently Asked Questions ==

= How to Configure this plugin? =

After install limopay plugin, at first you go to WooCommerce setting page. Then click Payments tab there you can see limopay payment gateway. Then click manage button where show many input fields such as (Shop id,Receiver UserName,Secret Key). This all information must be required.You get all information from limopay.net


= Does this require a Limopay merchant Accout? =

Yes! A Limopay merchant account, merchant shop id,Secret key , Public key ,Receiver Username,and x-api key are required for this gateway to function.

= How to Install Git Pass Apps from google playstore? =

 Go to google playstore, here you have to search- 'Gitpass bdtask' then install this apps  .

=How to enable Git Authenticator? =

After install 'gitpass' you need verify your gitpass app from limopay.net.

=Why Need WooCommerce Plugin? =

Without WooCommerce this plugin will not work.
